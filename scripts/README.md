# Scripts de génération de données

## Installation

```bash
pip install boto3
```

## Configuration AWS

```bash
aws configure
# Entrer vos credentials AWS
```

## Utilisation

```bash
python scripts/generate_test_data.py
```

- 5 produits (burgers, frites, boissons, desserts)
- 2 menus/formules

- 3 terminaux
